# AI-Tool-ISL-Generator
This project aims to develop an AI-powered mobile application that converts English/Hindi audio and video content into Indian Sign Language (ISL) and converts ISL gestures back into text or speech. The system helps bridge communication gaps between hearing-impaired and hearing individuals.

PROBLEM STATEMENT :

Many hearing-impaired individuals use Indian Sign Language (ISL) for communication. However, most educational, social, and digital content is available in spoken or written languages such as English and Hindi. This creates communication barriers between deaf and hearing communities. The proposed AI-based system converts audio-visual content into ISL and vice versa, enabling inclusive communication and accessibility.

Video Input Processing,
English/Hindi to ISL Conversion,
ISL Gesture Recognition,
ISL to Text Conversion,
Text to Speech Output,
History Management,
Report Generation.

Non-Functional Requirements :

Fast Response Time,
High Accuracy,
User-Friendly Interface,
Secure Data Storage,
Mobile Compatibility.

OBJECTIVES :

Convert English/Hindi speech into ISL animations.
Recognize ISL gestures through camera input.
Translate ISL gestures into text and speech.
Improve communication accessibility.
Provide a user-friendly mobile application.
